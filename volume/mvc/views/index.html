CONTENEDORES

Los contendedores son ambientes de ejeucción previamiente configurados los cuales tienen archivos, variables y librerías que son necesarias para funcionar. De esta manera la aplicación que desarrollaremos con el uso de un contenedor, se ejecutara de manera correcta aun cuando cambie su entorno, con esto facilitando su portabilidad.


Los barcos de tranporte utilizan contenedores para para organizar, acceder y entregar sus mercancias de forma más fácil y eficiente, al utilizar contenedores hacemos una organización de proyectos que utilizan diferentes herramientas de desarrollo para su funcionamiento, además, podemos realizar una reautilizacion de los mismos para proyectos futuros.
